# Obsidian

## Brief Summary
로컬 마크다운(.md) 파일을 관리하는 노트 앱. 모든 데이터가 내 컴퓨터에 저장되어 벤더 락인 없고, LLM이 직접 읽고 쓸 수 있다.

## Core Content
- 로컬 .md 파일 = AI가 바로 접근 가능
- wikilink (`[[노트이름]]`)로 노트 간 연결 + 그래프 뷰 시각화
- 1000+ 커뮤니티 플러그인 (Dataview, Templater, Git 등)
- Git으로 버전관리 + 백업 가능
- 노션/에버노트와 달리 벤더 락인 없음
- 우리 환경: ~/vault/ 를 vault로 열어서 wiki/ 탐색
