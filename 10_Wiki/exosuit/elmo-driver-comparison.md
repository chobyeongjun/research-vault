## 착용형 후보 3종 상세

### Gold Twitter (G-TWI) — 최소형 PCB 모듈

```
크기: 35 × 30 × 11.5 mm
무게: 18g (히트싱크 없음) / 39.6g (히트싱크 포함)
4개 합산: 72g ~ 158g
```

| 항목 | 내용 |
|------|------|
| 형태 | **PCB 직접 납땜** → 커스텀 보드 설계 필수 |
| 전압 | 100V 계열: 10~95V / 200V 계열: 20~195V |
| 연속 전류 | 모델별 1~50A |
| CANopen | DS301 + CiA 402 (CSP/CSV/CST/PP/PV/IP) |
| 단일 전원 | 'P' suffix 또는 'S' 버전 |
| 재생에너지 | 내부 shunt ⚠️ (크기 제약으로 용량 제한 가능) |
| 커넥터 | 없음 (PCB 핀) |

**장점**: 세계 최소 서보드라이브, 보드 통합 시 공간 최적화  
**단점**: PCB 설계 역량 필요, 납땜 공정 복잡

---

### Gold Solo Twitter (G-SOLTWI) — 독립 커넥터형

```
크기: ~47 × 40 × 30 mm
무게: 32~60g (모델별)
4개 합산: 128~240g
```

| 항목 | 내용 |
|------|------|
| 형태 | **독립 보드 + 커넥터** → 커스텀 PCB 불필요 |
| 전압 | 100V 계열: 10~95V / 200V 계열 |
| 연속 전류 | 최대 70A@100V |
| CANopen | 동일 |
| 단일 전원 | ✅ |
| 재생에너지 | 내부 shunt |
| 커넥터 | Molex 방식 (30핀 피드백 포함) |

**장점**: 커스텀 PCB 없이 케이블 연결로 즉시 사용, 프로토타입에 유리  
**단점**: Twitter 대비 크고 무거움

---

### Gold Whistle (G-WHIS) — PCB 모듈형 (I/O 풍부)

```
크기: 55 × 46 × 15 mm
무게: 55g
4개 합산: 220g
```

| 항목 | 내용 |
|------|------|
| 형태 | PCB 납땜 모듈 |
| 전압 | 12~95V DC (단일 전원) |
| **연속 전류** | **최대 20A** ← 한계 확인 필요 |
| CANopen | 동일 |
| 단일 전원 | ✅ (기본 설계) |
| I/O | 6DI / 2DO / 1AI → Twitter 대비 풍부 |
| 재생에너지 | 내부 shunt |

**장점**: 단일 전원 기본, I/O 풍부, 전압 범위 넓음(12~95V)  
**단점**: 최대 20A → 고전류 모터 제한, Twitter 대비 크고 무거움

## 결정 기준

### PCB 직접 설계 가능한가?

```
YES → Gold Twitter (최소 무게, 최고 통합도)
NO  → Gold Solo Twitter (커넥터 연결, 프로토타입 최적)
```

### 필요 전류가 20A 초과인가?

```
YES → Gold Twitter 또는 Solo Twitter (최대 50~70A)
NO  → Gold Whistle도 가능
```

### 배터리 전압?

```
24V (단일): Gold Whistle 가장 유연 (12~95V)
48V (직렬): 3종 모두 OK
```

## ⚠️ 확인 필요 사항

- Gold Twitter 내부 shunt 용량: 케이블 드리븐 반복 재생에너지에 충분한지 Elmo에 문의 필요
- 실제 필요 전류 계산 후 모델 확정 (모터 선택 → 토크 → 전류 역산)
