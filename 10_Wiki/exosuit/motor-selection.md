# 모터 선택 — 케이블 드리븐 외골격

드라이버: [[elmo-gold-twitter]] (G-TWI15/100, 10~95V, 15A)  
구동 방식: 케이블 + 풀리 / 속도 우선 > 토크

---

## 선택 기준

| 항목 | 요구사항 |
|------|---------|
| 드라이버 호환 | Elmo Twitter 지원 인코더 필수 |
| 전류 | 연속 15A 이내 |
| 속도 | 높은 Kv (고속 우선) |
| 무게 | 최소화 (착용형) |
| 형태 | 팬케이크 or 소형 (관절부 패키징) |
| 역구동성 | 케이블 드리븐 특성상 backdrivability 중요 |

---

## 후보 비교

### 1순위: T-Motor U8 Lite KV85 ⭐

| 항목 | 스펙 |
|------|------|
| 전압 | 24~48V |
| Kv | 85 rpm/V |
| Kt | ~0.112 Nm/A |
| 최대 RPM (48V) | **~4,080 rpm** |
| 연속 토크 (15A) | ~1.68 Nm |
| 피크 토크 (30A) | ~3.36 Nm |
| 무게 | **~220 g** |
| 크기 | φ87 × 28 mm (팬케이크) |
| 인코더 | **없음 → AS5047P 추가 필요** |
| 가격 | ~$60~80 |

**선택 이유:**
- 팬케이크 형상 → 관절부 패키징 최적
- 48V에서 4,080 rpm → 케이블 속도 최대화
- Backdrivability 우수 (낮은 감속비)
- Elmo 15A 범위 내 충분한 토크

**필요 추가 작업:**  
`AS5047P` (14-bit 자기식 인코더, SPI→ABZ 출력) → PCB에 통합 또는 모터에 부착

---

### 2순위: CubeMars RI60 KV120

| 항목 | 스펙 |
|------|------|
| 전압 | 12~48V |
| Kv | 120 rpm/V |
| Kt | ~0.080 Nm/A |
| 최대 RPM (48V) | ~5,760 rpm |
| 무게 | ~150~200 g (추정) |
| 크기 | φ60 (인러너) |
| 인코더 | **Hall 센서 내장** → Elmo Port A 직결 |
| 가격 | ~$80~120 |

**선택 이유:**
- Hall 내장 → 즉시 구동 가능 (가장 간단한 구성)
- 고속 (5,760 rpm)
- 외골격 전용 설계

**주의:** Hall만으로는 정밀 위치제어 한계 → 추가 인크리멘탈 인코더 권장

---

### 3순위: Maxon EC-i 40 70W

| 항목 | 스펙 |
|------|------|
| 전압 | 24~48V |
| 정격 전류 | ~5.6A |
| Kt | ~0.033 Nm/A |
| 최대 RPM | ~25,000 rpm (감속 필수) |
| 무게 | ~345 g (인코더 포함) |
| 크기 | φ40 × 113 mm |
| 인코더 | ENX/MILE (Elmo 공식 호환) |
| 가격 | ~$400~600 |

**선택 이유:** 외골격 실적 가장 많음, Elmo와 검증된 조합  
**단점:** 비싸고 길이가 길어 패키징 불리

## 인코더 호환 정리

| 모터 | 기본 피드백 | Elmo 연결 방법 |
|------|-----------|--------------|
| T-Motor U8 Lite | 없음 | **AS5047P** (ABZ 출력) → Port A |
| CubeMars RI60 | Hall 내장 | Hall → Port A 직결 |
| Maxon EC-i 40 | ENX/MILE | Elmo 공식 호환, 직결 |

---

## AS5047P 인코더 (U8 Lite 사용 시)

| 항목 | 스펙 |
|------|------|
| 해상도 | 14-bit (16,384 steps/rev) |
| 출력 | SPI, ABI (증분), UVW |
| 전압 | 3.3V 또는 5V |
| 크기 | 7 × 7 mm (TSSOP-14) |
| 자석 | 6mm 직경 2극 자석 (모터 축 중앙 부착) |
| 가격 | ~$5~10 |

---

## 배터리 전압에 따른 성능

| 전압 | U8 Lite KV85 무부하 RPM | 케이블 속도 비율 |
|------|----------------------|--------------|
| 24V | 2,040 rpm | 기준 |
| 48V | **4,080 rpm** | **2배** |

→ **속도 우선 → 48V 선택 권장**
